# Markdown 是什么

Markdown 是一种**轻量级标记语言**，由约翰·格鲁伯（John Gruber）于 2004 年创建。它允许人们使用易读易写的纯文本格式编写文档，然后转换成有效的 HTML 文档。

## 为什么选择 Markdown

| 优点 | 说明 |
|------|------|
| 简单易读 | 纯文本格式，无需特殊软件即可打开阅读 |
| 易学易用 | 语法简洁，几分钟即可上手 |
| 跨平台 | 支持 Windows、Mac、Linux 等所有平台 |
| 广泛应用 | GitHub、知乎、CSDN、Notion 等都支持 |
| 格式转换 | 可轻松转换为 HTML、PDF、Word 等格式 |

## 常用语法

### 1. 标题

使用 `#` 符号表示标题，`#` 数量表示标题级别：

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

**效果：**

# 一级标题
## 二级标题
### 三级标题

---

### 2. 段落和换行

段落之间用空行分隔：

```markdown
这是第一个段落。

这是第二个段落。
```

**效果：**

这是第一个段落。

这是第二个段落。

**换行**：在行末添加两个空格，或使用 `<br>`

```markdown
这是第一行  
这是第二行
```

---

### 3. 字体样式

```markdown
**粗体文字**
*斜体文字*
***粗斜体文字***
~~删除线文字~~
`行内代码`
```

**效果：**

**粗体文字**

*斜体文字*

***粗斜体文字***

~~删除线文字~~

`行内代码`

---

### 4. 列表

#### 无序列表

使用 `-`、`+` 或 `*` 开头：

```markdown
- 苹果
- 香蕉
- 橙子

* 项目一
* 项目二
* 项目三
```

**效果：**

- 苹果
- 香蕉
- 橙子

#### 有序列表

使用数字加 `.` 开头：

```markdown
1. 第一步
2. 第二步
3. 第三步
```

**效果：**

1. 第一步
2. 第二步
3. 第三步

#### 嵌套列表

```markdown
1. 第一章
   - 第一节
   - 第二节
2. 第二章
   - 第一节
   - 第二节
```

**效果：**

1. 第一章
   - 第一节
   - 第二节
2. 第二章
   - 第一节
   - 第二节

---

### 5. 链接

```markdown
[链接文字](https://www.example.com)

[带标题的链接](https://www.example.com "鼠标悬停显示")

[引用式链接][1]

[1]: https://www.example.com
```

**效果：**

[链接文字](https://www.example.com)

---

### 6. 图片

```markdown
![图片描述](图片地址)

![带标题的图片](图片地址 "鼠标悬停显示")
```

**效果示例：**

```markdown
![Markdown Logo](https://markdown-here.com/img/icon256.png)
```

---

### 7. 引用

使用 `>` 符号：

```markdown
> 这是一段引用文字。
> 可以有多行。

> 嵌套引用
>> 第二层引用
>>> 第三层引用
```

**效果：**

> 这是一段引用文字。
> 可以有多行。

> 嵌套引用
>> 第二层引用
>>> 第三层引用

---

### 8. 代码块

#### 行内代码

使用单个反引号：

```markdown
使用 `print()` 函数输出内容
```

**效果：**

使用 `print()` 函数输出内容

#### 代码块

使用三个反引号，可指定语言：

````markdown
```python
def hello():
    print("Hello, World!")
    return True
```

```javascript
function hello() {
    console.log("Hello, World!");
    return true;
}
```
````

**效果：**

```python
def hello():
    print("Hello, World!")
    return True
```

```javascript
function hello() {
    console.log("Hello, World!");
    return true;
}
```

### 10. 表格

```markdown
| 表头1 | 表头2 | 表头3 |
|-------|-------|-------|
| 内容1 | 内容2 | 内容3 |
| 内容4 | 内容5 | 内容6 |

# 对齐方式
| 左对齐 | 居中对齐 | 右对齐 |
|:-------|:--------:|-------:|
| 内容   |   内容   |   内容 |
```

**效果：**

| 表头1 | 表头2 | 表头3 |
|-------|-------|-------|
| 内容1 | 内容2 | 内容3 |
| 内容4 | 内容5 | 内容6 |

| 左对齐 | 居中对齐 | 右对齐 |
|:-------|:--------:|-------:|
| 内容   |   内容   |   内容 |

### 11. 任务列表

```markdown
- [x] 已完成任务
- [ ] 未完成任务
- [ ] 另一个未完成任务
```

**效果：**

- [x] 已完成任务
- [ ] 未完成任务
- [ ] 另一个未完成任务

---

### 12. 脚注

```markdown
这是一个有脚注的句子[^1]。

[^1]: 这是脚注的内容。
```

---

### 13. 特殊符号转义

使用反斜杠 `\` 转义特殊字符：

```markdown
\* 这不是斜体 \*
\# 这不是标题
\[ 这不是链接 \]
```

**效果：**

\* 这不是斜体 \*

\# 这不是标题

\[ 这不是链接 \]

---

### 14. HTML 内嵌

Markdown 支持直接嵌入 HTML：

```markdown
<details>
<summary>点击展开</summary>

这是隐藏的内容。

</details>

<center>居中的文字</center>
```

**效果：**

<details>
<summary>点击展开</summary>

这是隐藏的内容。

</details>
